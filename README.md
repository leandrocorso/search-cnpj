## Instalação

```
yarn install - instala as dependências
yarn start - executa o projeto
```

## Executar os testes

```
yarn test
```

## Coverage
O relatório de cobertura dos testes está presente na aplicação e pode ser acessado na pasta: /coverage/lcov-report/index.html
